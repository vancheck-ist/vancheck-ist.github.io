Challenges faced



Main challenges that the team faced in the first part of the project:

  • Difficulties in understanding and implementing appboard communication mechanisms (D-Bus, bluez), leading to the change from C to JavaScript;
  • Problems with the need to code in previously unknown that programming languages (JavaScript, CSS, HTML);
  • Finding an accessible RFID reader that is also compatible with our requirements, and with available documentation online;
  • Getting the website to be responsive, and working in various platforms;
  • Balancing coursework with other commitments;
