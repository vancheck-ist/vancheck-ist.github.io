Board communication



We have already managed to achieve basic communication with the board through the Rasperry Pi. For this purpose, a command was sent to confirm the hardware and software version. This command was chosen for the test as it is a command that, by definition of the communication protocol, the response is guaranteed, and the same was what was predicted.


Contribution:

  • Francisco Rosa
  • Duarte Pardal

    • Time frame:

  • From: 14/04 to 24/04

    • The next step will be to first do the integration test with all the modules of our project (app, Raspberry Pi and RFID reader). After that, we develop the code to carry out communication between the reader and the RFID tags, with testing limitations, for now, due to lack of access to the material.
