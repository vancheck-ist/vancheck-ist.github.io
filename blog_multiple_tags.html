Reliable reading of multiple tags



We can now read multiple tags!

Our app has long accepted more than one tag in the found tag list. (After all, nobody needs a system to perform the inventory of a single item.) But we've been bottlenecked by a issue with the communication with the RFID board. With that issue fixed, we can now read at least 6 tags at once, which is how many we've been given.
